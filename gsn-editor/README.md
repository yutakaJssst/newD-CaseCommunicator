# GSN Editor

Goal Structuring Notation (GSN) を用いた安全論証図を描画・編集するためのWebアプリケーション。

## 🎯 概要

このプロジェクトは、既存のD-Case Communicatorをモダンな技術スタックで再実装したものです。
まずは基本的なGSN描画機能のMVP（Minimum Viable Product）を実装しています。

## ✨ 実装済み機能

### Phase 1 (MVP)
- ✅ GSN要素（ノード）の描画
  - Goal（ゴール）
  - Strategy（戦略）
  - Context（前提）
  - Evidence（証拠）
- ✅ ノードの配置（クリックで追加）
- ✅ ノードの移動（ドラッグ&ドロップ）
- ✅ ノード間のリンク作成
- ✅ キャンバスのパン操作（Shift+ドラッグ）
- ✅ キャンバスのズーム操作（マウスホイール）
- ✅ JSONエクスポート/インポート
- ✅ ノード削除・リンク削除

## 🚀 クイックスタート

### 必要要件
- Node.js 18+
- npm

### インストール

\`\`\`bash
npm install
\`\`\`

### 開発サーバーの起動

\`\`\`bash
npm run dev
\`\`\`

ブラウザで \`http://localhost:5173/\` を開く

## 📖 使い方

### 1. ノードの追加
1. 左サイドバーからノードタイプを選択
2. キャンバス上の任意の場所をクリック

### 2. ノードの移動
- ノードをドラッグ&ドロップ

### 3. リンクの作成
1. 左サイドバーの「リンク追加」ボタンをクリック
2. 始点ノードをクリック
3. 終点ノードをクリック

### 4. キャンバス操作
- **パン**: Shift + ドラッグ
- **ズーム**: マウスホイール
